- Description: "Sync KMP ecosystem documentation to a unified Obsidian vault. Modes: init (create vault with .obsidian config), sync (update existing vault), clean (remove orphaned files)."
- Input schema (Zod):
  - `mode`: z.enum(["init", "sync", "clean"]).default("sync") -- operation mode
  - `vault_path`: z.string().optional() -- override vault path from config
- Handler:
  1. Check rate limit
  2. Build configOverride from vault_path if provided
  3. Switch on mode: init -> initVault(override), sync -> syncVault(override), clean -> cleanOrphans(override)
  4. Return JSON result: `{ mode, result: SyncResult, message: "Vault {mode} complete: {written} written, {unchanged} unchanged, {removed} removed" }`
  5. Wrap in try/catch: on error return `{ content: [{ type: "text", text: JSON.stringify({ error: message }) }] }`
