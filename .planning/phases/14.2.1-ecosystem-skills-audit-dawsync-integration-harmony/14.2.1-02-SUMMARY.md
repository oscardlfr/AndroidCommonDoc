## Decisions Made
- **Side-effect classification:** 8 skills restricted with disable-model-invocation: sync-vault, generate-rules, doc-reorganize, monitor-docs, sbom, sbom-scan, sbom-analyze, ingest-content. These all modify filesystem, vault, or external state.
- **Knowledge skill classification:** 4 web skills marked user-invocable: false (best-practices, core-web-vitals, performance, seo) as they provide background reference knowledge, not executable workflows.
- **Executable web skills:** accessibility and web-quality-audit kept user-invocable because they define actionable audit workflows.
- **Tool restrictions:** validate-patterns gets only Read/Grep/Glob (no Bash) since it uses agent built-in tools only. auto-cover gets Write/Edit in addition to standard tools since it generates test code.
- **Description format:** All descriptions follow "Action summary. Use when asked to [trigger conditions]." pattern for Claude's auto-discovery.
