**Changes to transformSource():**

**Slug disambiguation strategy (per research recommendation):**
- L0 pattern docs: use bare slug (e.g., `testing-patterns`) -- these are "canonical" names
- L1/L2 files: prefix slug with project name to prevent collisions (e.g., `shared-kmp-libs-TESTING_STRATEGY`, `DawSync-CLAUDE`)
- Exception: L1 pattern overrides from `.androidcommondoc/docs/` should use the original slug + `-L1-{project}` suffix to show override relationship
- Implementation: derive slug from relativePath basename for L0, add project prefix for L1/L2:
  ```typescript
  const baseSlug = path.basename(source.relativePath, ".md");
  const slug = source.layer === "L0" ? baseSlug : `${source.project ?? "unknown"}-${baseSlug}`;
  ```

**Changes to transformAll():**
- Build allSlugs set using the new disambiguation logic
- Ensure slug calculation is consistent between the allSlugs set and individual transformSource calls

**TagGeneratorParams extension (pass to generateTags):**
- Add `subProject?: string` to the params passed to generateTags
